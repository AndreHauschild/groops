GnssStationInfo (file format)

Information file for a station or satellite equipped with a GNSS receiver or for a GNSS satellite.

For stations and receiving satellites it defines the time intervals and local coordinates of installed antennas and receivers. For GNSS satellites it defines the PRN-SVN mapping and antenna offsets and orientations.

GNSS station infos can be created from station log files with GnssStationLog2StationInfo. GNSS satellite transmitter infos can be created from an ANTEX file with GnssAntex2AntennaDefinition.

See GnssStationInfoCreate.
